# jehwan-react-template

## Stack

- FE: React, TypeScript, Tailwind CSS, Heroicons, Headless UI
- Dev: Husky, Jest, Testing Library, ESLint, Prettier

## Installation

### Package

```bash
npm install
```

### Git Hook

```bash
npm run prepare
```

## Start

```bash
npm start
```

## Build

```bash
npm run build
```

## Test

### Watch Mode

```bash
npm run test
```

### No Watch Mode

```bash
npm run test-no-watch
```

## Code Style

### Lint

```bash
npm run lint
```

### Fix & Format

```bash
npm run fix
```
